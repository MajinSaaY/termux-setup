# Termux-setup
[English](https://github.com/MajinSaaY/termux-setup)

Minha configuração pessoal do Termux, nada especial...
Ele baixa vídeos, músicas, fotos, torrents, etc.

# Pré-requisito
- [Termux](https://f-droid.org/packages/com.termux/)
- [Termux:API](https://f-droid.org/pt_BR/packages/com.termux.api/) (opcional)
- [Termux:Widget](https://f-droid.org/pt_BR/packages/com.termux.widget/) (opcional)
- [Termux:Styling](https://f-droid.org/pt_BR/packages/com.termux.styling/) (opcional)

# Instalação
Copie e cole no seu Termux:
```
termux-setup-storage && bash <(curl https://raw.githubusercontent.com/MajinSaaY/termux-setup/main/setup_PT-BR.sh)
```

# Uso
Compartilhe qualquer URL para o Termux:
